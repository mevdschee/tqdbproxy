# System Components

TQDBProxy is built using a modular architecture. Below are the core components that make up the system:

## Component Index

- **[Cache](cache/README.md)**: Query result caching using Otter.
- **[Metrics](metrics/README.md)**: Prometheus metrics collection.
- **[MySQL Protocol](mysql/README.md)**: MySQL wire protocol implementation.
- **[SQL Parser](parser/README.md)**: SQL hint and metadata extraction.
- **[Generic Proxy](proxy/README.md)**: Base TCP proxy implementation.
- **[Replica Management](replica/README.md)**: Database pool and health checks.
